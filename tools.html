Tools

Ecore4Reg collaborative content creation tools

Eclipse Free BIRD Tools provides open source implementations of:
  • Text editors for the file formats with auto-completion
  • Visual and textual navigation of the artefacts
  • Data aware forms for the creation of test data

Connectors

Eclipse Free BIRD Tools provides open source connectors written in Python which take as input the artefacts (such as the Data Model) produced by the BIRD project and translates them to Ecore4Reg format. Connectors could be provided to other open or closed regulatory data models also.
