BIG Open Studio

BIG Open Studio is a software application with guided tooling for generating the Free BIRD Platform. The studio runs on the desktop using mature open source technologies, specifically The Eclipse IDE , and its plug-in architecture for creating and installing add-ons. Like all BIG Studios it allows users to:
  • Create the BIRDseed from BIRD artefacts
  • Edit and Navigate BIRDSeeds in text form or visual form (such as an ER diagram)
  • Run a BIG Pot to generate a version of the data-models and transformations suitable for the target platform (the Free BIRD Platform in this case)
  • Create the complete usable configurations for the target platform
