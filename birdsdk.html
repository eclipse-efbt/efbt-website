BIRD Software Developement Kit for Deskto applications

A tested toolkit for writing apps using BIRD


  • BIRD SDK provides a Java and Python API for interacting with the BIRD datamodel and serializing data to and from files
  • Write desktop applications using BIRD quickly using the API...no need to start from zero
  • BIRD SDK is autogenerated by the Eclipse Free BIRD Tools project from RegDNA files
  • BIRD SDK provides a Java and Python class for each Table/Entity in the BIRD Input Layer and Logical DataModel and retains all relationships
  • BIRD SDK uses the Ecore Framework, and works well within the Eclipse Rich Client Platform, and can interact easily with existing components from both.
  • BIRD SDK works with the BIRD Input Layer and Logical Data Model
