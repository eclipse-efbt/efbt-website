Quickstart

Get up and running with Eclipse Free BIRD Tools in minutes using our simple installation scripts.

For Windows (and Servers):

windows_startup.bat

Otherwise:

chmod +x linux_startup.sh
./linux_startup.sh

Note: The linux_startup.sh script works for both Linux based and macOS systems.
