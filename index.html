Task Focussed

The primary goal of Eclipse Free BIRD Tools is to be a task oriented environment , providing the maximum tool support possible in completing the common time consuming tasks to be done while developing, changing , or using a regulatory calculation engine written using VTL and SMCubes, such as BIRD.

Component Based

Being components based it is possible to choose just the components that you need, maybe you want your own visualization, but want to re-use the VTL engine, that is fine.

Open Source

Being open source these components improve over time , are openly available to all, and are published under a commercially friendly license allowing software vendors to add value and provide support. Why re-invent the wheel?
